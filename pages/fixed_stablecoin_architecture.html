1
선불 충전 (Pre-funding)
사용자가 기존 간편결제 앱과 동일한 방식으로 스테이블코인을 퍼실리테이터에게 예치하여 포인트를 충전합니다.
기술적 구현
KRW → USDC 변환, 퍼실리테이터 계정에 예치, 사용자 잔액 업데이트
2
결제 승인 (Authorization)
사용자가 가맹점에서 결제 시도 시, 기존 PG사 시스템과 동일한 방식으로 PIN이나 생체인증을 거쳐 결제 승인을 처리합니다.
보안 검증
다중 인증, 잔액 확인, 사기 탐지 시스템(FDS) 연동
3
VASP 정산 요청 (/settle)
결제가 최종 확정되면 퍼실리테이터가 VASP에게 정산을 요청합니다. 이때 고빈도 API 호출이 발생합니다.
API 스펙
RESTful API, 실시간 처리, VISA/Mastercard 수준의 처리량 지원
4
KYC/AML/Travel Rule 검증
VASP가 수취인(가맹점)에 대한 KYC 확인 및 트래블룰 준수를 위한 정보 수집을 실시간으로 처리합니다.
규제 준수
특금법 준수, 100만원 이상 거래 시 정보 전달, AML 모니터링
5
온체인 트랜잭션 실행
VASP가 보관 지갑에서 가맹점 지갑으로 해당 금액의 스테이블코인을 블록체인 상에서 전송합니다.
블록체인 처리
스마트컨트랙트 실행, 가스비 최적화, 트랜잭션 확인
6
결제 완료 확인
온체인 트랜잭션이 확인되면 모든 참여자에게 결제 완료 알림이 전송되고 최종 정산이 완료됩니다.
완료 처리
성공/실패 응답, 영수증 발행, 회계 처리, 정산 완료

🛡️ 핵심 규제 준수 요소

  • 특정금융정보법: VASP 라이선스 기반 KYC/AML 완전 준수
  • 전자금융거래법: 퍼실리테이터의 PG사 라이선스 필수
  • 가상자산이용자보호법: 분리보관 및 소비자 보호
  • 외국환거래법: 국경간 거래 시 신고 의무 자동화
  • 실명인증: 주민등록번호 기반 본인확인 연동
  • 고빈도 처리: VISA/Mastercard 수준의 TPS 지원
